### How to run

1. Clone the repo
   ```sh
   git clone https://github.com/kumancev/multifarm-chart.git
   ```
2. Install Dependencies
   ```sh
   npm install
   ```
3. Run Development Enviroment
   ```sh
   npm run dev
   ```
4. Build Project
   ```sh
   npm run build
   ```
